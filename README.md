# App Contable Cotizaciones Offline

App movil offline con Expo, React Native y SQLite local.

Flujo principal:

Cotizacion creada -> Cotizacion confirmada -> Venta registrada.

Modulos iniciales:

- Inicio
- Cotizaciones
- Ventas
- Clientes
- Inventario
- Movimientos

Para instalar:

npm install

Para probar:

npx expo start
